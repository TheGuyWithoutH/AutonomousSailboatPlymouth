Schematics

Wiring Table containing the wiring map This table is also useful for all the code

!!! WARNING !!! As there are some expensive components on the boat, you'd better check twice that all wiring is correct before uploading the code. Moreover, for the first time it is hardly recommended to upload the code first (with a plain Arduino Mega) then to unplug the power supply, to link every sensor / actuators and finally to plug in the power wire
